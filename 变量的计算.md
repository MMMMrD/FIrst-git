# 二. 变量的计算



## 1.计算过程

> 一个变量既作为某计算过程(加/减)的过程量，又作为其过程的结果量，则可将计算式简化为
>
> ~~~
> x -= y;
> x += y;
> ~~~
>
> 

## 2.计算结果

>(1)可由Debug.Log()直接输出
>
>(2)可将结果赋给其他变量输出(被赋值的变量的类型一定要与结果的类型相同)
>
>(3)要在输出结果时同时输出字符，则
>
>~~~
>public int x;
>public int y;
>Debug.Log(x+" 's health is "+y);
>~~~
